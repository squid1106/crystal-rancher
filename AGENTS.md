# Crystal Rancher — Project Instructions

Read the master development specification before changing game systems.

- Keep monsters, jobs, affinities, buildings, recruits, and abilities data-driven.
- Keep simulation logic independent from React and Phaser rendering.
- Use original art only. Never download or embed copyrighted third-party game assets.
- Do not implement systems beyond the current tutorial milestone without approval.
- Every game-system change requires tests. Run both `npm test` and `npm run build`.
- Save data must be versioned and migrated deliberately.
