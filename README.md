# Crystal Rancher

A cozy, mobile-friendly pixel-art monster sanctuary RPG. Build the ranch, care for creatures, recruit adventurers, and explore Briarglen Forest.

## Local development

```sh
npm install
npm run dev
```

## Verification

```sh
npm test
npm run build
```

## Hosting

- Frontend: Vercel
- Backend foundation: Supabase
- Source: GitHub

The current playable save remains browser-local. Supabase is provisioned for the future authenticated cloud-save milestone; no unsafe public tables are exposed.
